# RefTracker 자막 프록시 서버

RefTracker 웹앱의 "자막" 기능이 실제로 작동하려면 이 서버가 인터넷 어딘가에서 돌아가고 있어야 합니다.
아래 방법대로 배포하면, **어느 컴퓨터에서 RefTracker를 열든** 같은 자막 서버 주소를 설정에 입력해서 쓸 수 있습니다.

---

## 방법 1: Render.com에 무료 배포 (추천, 신용카드 불필요)

1. https://render.com 에 접속해서 GitHub 계정으로 가입/로그인
2. 이 폴더(`transcript-proxy`)를 본인 GitHub 저장소에 업로드
   - GitHub 웹사이트에서 새 저장소(New repository)를 만들고, `server.js`, `package.json`, `README.md` 세 파일을 업로드하면 됩니다.
3. Render 대시보드에서 **New + → Web Service** 클릭
4. 방금 만든 GitHub 저장소 선택
5. 설정값 입력:
   - **Name**: 아무 이름 (예: reftracker-transcript)
   - **Region**: 가까운 지역 아무거나 (Singapore 추천)
   - **Branch**: main
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Instance Type**: Free
6. **Create Web Service** 클릭 → 2~3분 기다리면 배포 완료
7. 배포가 끝나면 화면 상단에 주소가 생깁니다. 예: `https://reftracker-transcript.onrender.com`
8. RefTracker 설정 → 자막 프록시 → 아래 주소를 입력하고 저장:
   ```
   https://reftracker-transcript.onrender.com/transcript
   ```

⚠️ 무료 요금제는 15분간 요청이 없으면 서버가 잠들어서, 다시 요청하면 첫 응답이 30초~1분 정도 걸릴 수 있습니다. 정상입니다.

---

## 방법 2: Railway.app에 배포

1. https://railway.app 가입 (GitHub 계정으로 로그인 가능)
2. New Project → Deploy from GitHub repo → 이 폴더를 올린 저장소 선택
3. 자동으로 `npm install` → `npm start` 로 배포됩니다
4. 배포된 도메인을 Settings → Networking → Generate Domain 에서 생성
5. 생성된 주소 + `/transcript` 를 RefTracker 설정에 입력

---

## 방법 3: 로컬에서만 테스트 (다른 컴퓨터에서는 접속 불가)

```bash
npm install
npm start
```

`http://localhost:3000/transcript` 로 접속됩니다. 이 방식은 내 컴퓨터에서만 열립니다.
다른 컴퓨터에서도 쓰려면 방법 1이나 2로 실제 배포해야 합니다.

---

## 동작 확인 방법

배포한 주소가 `https://xxxx.onrender.com` 이라면, 브라우저에서 아래 주소로 들어가서 자막 데이터가 JSON으로 나오는지 확인하세요.

```
https://xxxx.onrender.com/transcript?videoId=dQw4w9WgXcQ
```

정상이라면 아래와 비슷한 JSON이 보입니다:
```json
{
  "language": "en",
  "isAutoGenerated": true,
  "segments": [
    { "start": 0, "duration": 3.5, "text": "..." }
  ]
}
```

## 참고 사항

- 이 서버는 유튜브가 공개적으로 제공하는 자막 데이터를 가져올 뿐, 별도 인증키가 필요 없습니다.
- 유튜브 쪽에서 자막을 막아둔 영상(자막 비활성화)은 이 서버로도 가져올 수 없습니다. 이 경우 RefTracker 화면에 "자막을 찾을 수 없습니다" 같은 오류가 정상적으로 표시됩니다.
- 무료 호스팅은 트래픽이 많아지면 느려지거나 제한이 걸릴 수 있습니다. 개인 테스트/소규모 사용 용도로 충분합니다.
